「鹿肉カレー」 ~森と共に生きる~

地元の食材と人にふれて心と身体を育てる料理教室です。

第一回目は、鹿肉と地元の野菜を使ってカレーを作ります。

料理技術と食にまつわる問題や知恵やマナーを丁寧に学びます。

衛生責任者資格を持つ料理インストラクター指導のもと、 地元の料理サークルの大学生たちが先生となって教えます。
